# My first Data Analyst project! And I built it entirely in Excel.

In just one day, I worked on an Exploratory Data Analysis (EDA) on supermarket sales data, with the goal of transforming raw data into actionable business insights.

### 📊 The result? An interactive dashboard that includes:
• Key KPIs: $2.26M in total sales and best month identified
• Top 10 products by sales and Top 5 by volume ranking
• Category and subcategory analysis with dynamic slicers
• Monthly sales trend to identify seasonality

### 🔍 Some insights from the analysis:
• Technology is the highest-revenue category
• November 2018 was the best month in sales
• Office products lead in transaction volume

### Tools used: Microsoft Excel (Pivot Tables, Charts, Slicers, INDEX, XMATCH, FILTER, Conditional Formatting)

This project allowed me to develop skills in data cleaning, visualization, and business-oriented analysis.

Dataset used:
https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting
